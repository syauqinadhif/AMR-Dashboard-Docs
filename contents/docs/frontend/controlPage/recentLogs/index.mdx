---
title: Recent Logs Component
description: Komponen ini menampilkan log terbaru dengan waktu dan tingkat keparahan (severity). Pengguna dapat melihat pesan log dan mengidentifikasi tingkat keparahan berdasarkan ikon berwarna.
---

## Fitur Utama
1. **Menampilkan Log Terbaru**: Daftar log dengan pesan, waktu, dan tingkat keparahan ditampilkan dengan rapi.
2. **Indikator Keparahan**: Setiap log memiliki indikator warna untuk tingkat keparahan: biru untuk `info`, kuning untuk `warning`, dan merah untuk `error`.
3. **Format Waktu**: Waktu setiap log ditampilkan dalam format 24 jam dengan jam, menit, dan detik.
4. **Tombol Filter**: Tombol filter disediakan untuk aksi lebih lanjut (meskipun belum terimplementasi).

## Teknologi yang Digunakan
- **Vue.js 3** (Composition API)
- **PrimeVue** untuk komponen tombol (`Button`).
- **Tailwind CSS** untuk styling.

function formatTime(timestamp: number): string {
