---

## Fitur Utama
1. **Menampilkan Detail Agen**: Menampilkan informasi dasar tentang agen seperti nama, situs, dan status.
2. **Edit Agen**: Memungkinkan pengguna untuk mengedit nama, alamat IP, HSM Key, dan register plant agen melalui dialog.
3. **Periksa Status Port**: Dialog untuk menampilkan status port agen menggunakan komponen `AgentPortStatus`.
4. **Navigasi Status dan Kontrol**: Menyediakan navigasi menuju halaman status agen dan kontrol agen.

## Teknologi yang Digunakan
- **Vue.js 3** dengan Composition API
- **PrimeVue** untuk komponen UI seperti `Dialog`, `InputText`, dan `Button`.
- **Vue Router** untuk navigasi melalui `RouterLink`.
- **Custom Components** seperti `AgentStatus`, `AgentPortStatus`, `StatusIcon`, `EditIcon`, dan `MapAndControlIcon`.
