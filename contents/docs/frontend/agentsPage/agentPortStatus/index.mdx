## Fitur Utama
1. **Menampilkan Status Port**: Setiap port akan ditampilkan dengan status apakah terhubung atau tidak.
2. **Indikator Visual**: Status port ditampilkan dengan indikator warna: hijau untuk terhubung dan merah untuk tidak terhubung.

## Teknologi yang Digunakan
- **Vue.js 3** (Composition API)
- **Tailwind CSS** untuk styling warna dan layout.
