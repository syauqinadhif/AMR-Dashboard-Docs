---
title: Agent Status Card
description: Komponen `AgentStatusCard.vue` menampilkan informasi tentang agen tertentu, seperti status baterai, kecepatan linear dan angular, serta kekuatan sinyal. Komponen ini menggunakan Vue.js dengan Composition API untuk mengelola data agen secara reaktif.
---

## Fitur Utama
1. **Indikator Baterai**: Menampilkan status baterai agen dalam bentuk diagram progress circular dengan persentase.
2. **Kecepatan Linear dan Angular**: Menampilkan kecepatan linear (m/s) dan kecepatan angular (rad/s) agen.
3. **Indikator Sinyal**: Menampilkan kekuatan sinyal agen dengan ikon khusus.
4. **Siklus Pemeliharaan**: Menampilkan informasi siklus pemeliharaan dalam satuan jam.

## Teknologi yang Digunakan
- **Vue.js 3** (Composition API) untuk manajemen state dan komputasi reaktif.
- **PrimeVue** untuk elemen UI seperti `SelectButton` (di luar contoh ini).
- **Komponen Khusus**:
  - `SignalIcon.vue`: Menampilkan ikon kekuatan sinyal.
  
## Potongan Kode `AgentStatusCard.vue`

### Script Section

```typescript
<script setup lang="ts">
import { computed } from 'vue'
import SignalIcon from '@/components/icon/SignalIcon.vue'
import { Agent } from '@/types/agent'

// Mengambil properti agen
const props = defineProps<{
  agent: Agent;
}>();

// Hitung status baterai agen
const agentBattery = computed(() => props.agent.battery ?? 0);
</script>
```

### Template Section
```html
<template>
  <div class="flex-col justify-center w-72 h-96 bg-white shadow-md p-4 rounded-lg">
    <h1 class="font-semibold text-2xl text-primaryblue text-center">
      {{ agent.name }}
    </h1>

    <!-- Indikator Baterai -->
    <div class="text-center mt-4">
      <v-progress-circular
        :model-value="agentBattery"
        :rotate="360"
        :size="100"
        :width="9"
        class="v-progress-circular"
        :class="agentBattery <= 20 ? 'text-primaryred' : 'text-primaryblue'"
        :style="{ strokeLinecap: 'round' }"
      >
        <template v-slot:default>
          <span class="font-semibold text-primaryblue-500">{{ agentBattery }}%</span>
        </template>
      </v-progress-circular>
    </div>

    <!-- Kecepatan Linear & Angular -->
    <div class="mt-6">
      <div class="flex justify-between w-full px-4 mb-1">
        <p class="font-semibold text-primaryblue mt-1">
          {{ (agent.linearVelo ?? 0).toFixed(2) + " m/s" }}
        </p>
        <p class="font-semibold text-primaryblue mt-1">
          {{ (agent.angularVelo ?? 0).toFixed(2) + " rad/s" }}
        </p>
      </div>
      <div class="flex justify-between w-full px-4 text-sm">
        <h2 class="font-semibold text-primaryblue">Linear Vel.</h2>
        <h2 class="font-semibold text-primaryblue">Angular Vel.</h2>
      </div>
    </div>

    <!-- Indikator Sinyal dan Siklus Pemeliharaan -->
    <div class="flex gap-x-8 p-2 justify-center ml-6">
      <div class="text-center" title="Signal Strength">
        <SignalIcon alt="Signal Icon" class="h-8" />
        <p class="font-semibold text-primaryblue">Signal</p>
      </div>

      <div class="text-center" title="Maintenance Cycle">
        <h2 class="font-bold text-2xl text-primaryblue">{{ 12 }}H</h2>
        <p class="font-semibold text-primaryblue mb-4">Mnt. Cycle</p>
      </div>
    </div>
  </div>
</template>
